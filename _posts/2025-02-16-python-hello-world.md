这一篇是 Python 的 **Hello World** 示例，用来测试博客的代码高亮和文章结构。

最经典的 Python Hello World 代码只有一行：

```python
print("Hello, World")
```

如果你在命令行中运行：

```bash
python hello.py
```

就会在终端看到：

```text
Hello, World
```
